# Random Task Generator

**Автор**: [Ваши Имя и Фамилия]

Консольное приложение на Python, позволяющее генерировать случайные задачи
из пула, фильтровать их по типу и сложности, вести историю (очередь) и
сохранять/загружать историю в JSON.

Архитектура построена на объектно-ориентированных принципах с использованием
паттерна «Фабрика» (Factory Method).

## Возможности

- Добавление новых задач в пул (type: work/sport/study, difficulty: easy/medium/hard)
- Генерация случайной задачи с опциональными фильтрами по типу и/или сложности
- Просмотр истории всех сгенерированных задач (FIFO)
- Сохранение истории в JSON-файл
- Загрузка истории из ранее сохранённого JSON
- Валидация пользовательского ввода
